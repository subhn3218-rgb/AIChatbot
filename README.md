# 🤖 ARiA Pro — Smart AI Chatbot Platform

**ARiA Pro** is a feature-rich, interactive AI chatbot web application designed with a sleek UI/UX, robust user authentication, and advanced customizable training features. It allows users to converse with an AI, train the bot with custom knowledge, and manage their account and plans effortlessly.

---
## ✨ Key Features

### 🔐 1. User Authentication (Secure Access)
*   **Sign Up & Log In:** A modern, secure screen for user registration and authentication to protect user data and personalize their experience.
*   **Sign Out:** Seamlessly logs users out of their active session, redirecting them securely back to the login page.

### 💬 2. Chat Workspace & Navigation
*   **New Chats:** Create multiple chat sessions dynamically to organize your conversations with the AI.
*   **Collapsible Sidebar:** A responsive sidebar that keeps your chat history, settings, and navigation links neat and clean.

### 🧠 3. Train Bot (Custom Q&A Training)
*   **Self-Training Assistant:** If the chatbot encounters a question it doesn't know, you can directly train it!
*   **Dynamic Q&A Submission:** Easily input a specific question alongside its correct answer to expand the bot's custom knowledge base.

### 💳 4. Subscription & Upgrades
*   **Subscription Tier Page:** An organized pricing and upgrade layout displaying available tiers:
    *   **Free Plan** (Basic AI features)
    *   **Plus Plan** (Advanced features and priority support)
    *   **Pro Plan** (Full features, custom training, and highest performance)

### ⚙️ 5. My Account & Settings
*   **Upgrade Plan Settings:** Convenient access to change or upgrade your current active subscription plan.
*   **General Settings:** Customize application behaviors and personal preferences.

---
## 🛠️ Tech Stack & Technologies

*   **Frontend:** HTML5, CSS3, JavaScript (Modern, responsive grid-based layouts)
*   **AI Integration:** Gemini API / Google AI Studio integration.

---

## 🚀 How It Works (Project Flow)

1.  **Authentication:** Users land on the **Login/Sign Up** screen to access the application.
2.  **Home Dashboard:** Users can start a conversation, create **New Chats**, or navigate through the sidebar.
3.  **Bot Training:** Go to the **Train Bot** section, input a Question and its corresponding Answer, and save it. The bot will now use this data to respond accurately in the future.
4.  **Manage Settings:** Check current tiers in the **Upgrade** section, configure personal layouts in **Settings**, or **Sign Out** to end the session.
